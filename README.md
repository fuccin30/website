# Economics PhD Website Shells

This workspace contains three static website shells for an economics PhD student preparing for conferences, job-market visibility, and future professional use.

Open `index.html` in a browser to compare the options:

- `sites/minimal-job-market/`: a focused, one-page job-market style site.
- `sites/classic-academic/`: a traditional academic homepage with clear sections.
- `sites/modern-research/`: a polished research portfolio with stronger visual hierarchy.

Use `intake-checklist.md` to gather the information needed to personalize whichever version you choose.
